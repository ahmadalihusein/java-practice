# Project 1 — To-Do List App (console)

Your first COMPLETE app. It's not new magic — it's the pieces you already know,
assembled, plus one new thing: reading what the user types (Scanner).

## What it'll do when finished
A menu that loops until you quit:
```
=== MY TO-DO LIST ===
1. Add a task
2. View tasks
3. Mark task as done
4. Quit
Choose:
```

## How it maps to what you already know
- Each task = an OBJECT (like your Product / Dog)        ← 03-objects
- All tasks = an ArrayList of those objects              ← 02-data
- The menu = a loop + if/else on the user's choice       ← 01-basics
- NEW: Scanner = lets the user type input
